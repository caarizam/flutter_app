# Basic flutter app

## install dependencies
```flutter pub get```

## execute the app
```flutter run```

## execute the automated tests
- ```flutter drive --target=test_driver/app.dart```
- ```flutter drive --driver=test_driver/integration_test.dart --target=integration_test/login_test.dart```
